HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Semantic elements are elements that clearly described its meaning to both the browser and the developer. Examples:
    • <form>
    • <table>
    • <article>
  • Non-semantic elements tell nothing about its contents. Examples:
    • <div>
    • <span>
  • HTML element holds the content
  • HTML attributes are used to describe the characteristic of an HTML element in detail. They are only found in the starting tag

CSS

  • CSS: Cascading Style Sheets
  • There are 3 ways to style a webpage using CSS
    • inline CSS
    • internal CSS style sheet
    • external CSS style sheet
  • Margin property adds spacing around the outside of an element, whereas the padding property adds spacing to the inside of the element, thus making the element bigger.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

Javascript

  • Browser reads code in the order it appears in the file
  • Arrow function => ()
  • Anonymous function ()
  • falsy values:
    • 0, -0, On, "", null, undefined, NaN, or bool with value of False
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
